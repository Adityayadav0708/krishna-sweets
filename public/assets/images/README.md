# Image assets

Keep replaceable photography grouped by purpose: `products`, `categories`, `festivals`, `gallery`, and `brand`.

- Prefer AVIF with WebP fallback where practical.
- Record the original source and license in `sources.md`.
- Use descriptive lowercase filenames such as `kesar-pista-barfi-01.webp`.
- Do not encode business content into filenames or component markup; image references belong in the central data files.
