# Krishna Sweets

Premium frontend for a heritage Indian sweets house established in 1948.

## Commands

- `npm run dev` — start the local development server
- `npm run build` — type-check and create a production build
- `npm run lint` — run static analysis

## Content replacement

All business details and navigation content live in `src/data/site.ts`. Product and editorial data will use the same central-data pattern. Development image files belong in `public/assets/images`; their replacement and source notes are documented there.
